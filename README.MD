# 개인 목표 및 구상

```
백단을 스프링 부트로 작성 및
프론트단을 리액트로 연결

세팅 완료 후 간단한 CRUD 테스트 후
외부 API와 연동해서 아이디어를 더 생각해 본뒤 진행 예정
```
